# Tyria - Tourism Mobile Application

[![Flutter](https://img.shields.io/badge/Flutter-Mobile%20App-02569B.svg)](https://flutter.dev/)
[![Dart](https://img.shields.io/badge/Dart-2.x-0175C2.svg)](https://dart.dev/)
[![GetX](https://img.shields.io/badge/GetX-State%20Management-purple.svg)](https://pub.dev/packages/get)
[![Dio](https://img.shields.io/badge/Dio-4.0.4-orange.svg)](https://pub.dev/packages/dio)
[![Mapbox](https://img.shields.io/badge/Mapbox-Navigation-blue.svg)](https://www.mapbox.com/)
[![Flutter Map](https://img.shields.io/badge/Flutter_Map-1.0.0-green.svg)](https://pub.dev/packages/flutter_map)

A Flutter mobile application designed to help users explore tourist destinations, discover attractions, and plan visits through an intuitive mobile experience,
and also choosing the most efficiant route using Genetic Algorithm.

---

# Features

- User registration and authentication
- Browse tourist destinations
- Ratings and reviews display
- Navigation support using Mapbox
- Loading skeleton animations
---
## Technical Highlights
- **Flutter & Dart** for cross-platform mobile development
- **GetX** for state management, dependency injection, and routing
- **RESTful APIs** using Dio for data retrieval and backend communication
- **Mapbox** integration for maps and navigation
- **Location Services** for location-aware functionality
- **Shared Preferences** and **Get Storage** for caching
- **Flutter ScreenUtil** for responsive layouts
- **Shimmer** loading animations and enhanced user experience
